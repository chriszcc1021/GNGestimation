## 核心预估逻辑

### Install 来源分类

```
总 Install = 回流（Revival） + OM（买量） + Organic（自然）
```

三类 Install 均通过留存曲线贡献 DAU。

## 数学模型

### Install → DAU 转化公式

```
C = (1/30) × Σ_{t=1}^{30} Σ_{k=1}^{t} [ NF(k)×(1-rp) + RF(k)×rp ] × retention(t-k+1)

totalInstall = avgDAU / C
```

- `NF(k)` = 新设备到达分布（前7天60%，7-14天25%，14-30天15%）
- `RF(k)` = 回流到达分布（D1: 35%，D2: 25%，D3: 15%，D4-7: 5%均分，D7-14: 15%均分，D14-30: 5%均分）

### DAU ↔ PCU ↔ 等待时长（幂律回归）

```
PCU = a × DAU^b        （各区独立拟合）
Wait = a × PCU^b       （各区独立拟合）
```

反推：`targetWait → PCU → DAU`（用于匹配快/慢方案）
